# 🌐 Linktree - Projeto em React

<img src="/print.png">

Hospedagem do site: [Linktree](https://bit.ly/iamkleiton)


Este projeto é uma aplicação desenvolvida com **React.js** que simula uma página de links pessoais, como o Linktree. O objetivo é reunir todos os seus links importantes (redes sociais, portfólio, contatos, etc.) em uma única página responsiva e moderna.

## 🔥 Funcionalidades

- Interface clean e responsiva
- Ícones personalizados com Tabler Icons
- Estrutura modular com componentes reutilizáveis
- Estilização com CSS moderno
- Links externos para redes sociais e portfólio

## 🛠 Tecnologias Utilizadas

- [React.js](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Tabler Icons](https://tabler.io/icons)
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/kleitonmac/linktree-links.git

# Acesse o diretório
cd linktree-pages

# Instale as dependências
npm install

# Rode o projeto localmente
npm run dev
